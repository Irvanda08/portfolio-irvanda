Skripsi / IoT

Monitoring Kualitas Air (ANFIS)

Web Dashboard & Grafik

Dashboard ini dibuat menggunakan HTML, CSS (Tailwind), dan JavaScript untuk mengambil data secara real-time dari Firebase. Menggunakan Chart.js untuk memvisualisasikan tren suhu, pH, dan kekeruhan.

Tampilan Dashboard Utama

Logika Cerdas (ANFIS)

Sistem ini tidak hanya memonitor, tetapi juga melakukan pengendalian otomatis. Algoritma Adaptive Neuro-Fuzzy Inference System (ANFIS) diprogram menggunakan Python untuk menentukan kapan pompa atau heater harus menyala berdasarkan data sensor yang kompleks.

  • Fuzzifikasi Data Input (Suhu, pH, NTU)
  • Pembentukan Basis Aturan (Rule Base)
  • Defuzzifikasi Output (Durasi Aktuator)

Integrasi Hardware

Integrasi mikrokontroler (ESP32) dengan sensor pH, DS18B20 (Suhu), dan Turbidity Sensor. Data dikirim ke Firebase via WiFi.
