## ğŸ“ Showcase Proyek Utama

### 1. Inventory System PT Muara Mitra Mandiri (Real Case)
Sistem manajemen stok operasional yang digunakan oleh perusahaan untuk memantau mutasi barang secara mendetail dan real-time.
* **Fitur**: Dashboard statistik harian, pelaporan stok otomatis (Stok Awal + Masuk - Keluar), dan sistem kartu gudang digital.
* **Teknologi**: PHP Native, MySQL, Tailwind CSS.

### 2. Admin Stok Gudang (Versi Akademik)
Panel administrasi inventaris dengan fokus pada manajemen database dan pelaporan dokumen.
* **Fitur**: CRUD barang lengkap, filter laporan berdasarkan periode (7 hari, 1 bulan, dll), dan fitur cetak laporan ke format PDF menggunakan library **FPDF**.
* **Teknologi**: PHP, Bootstrap 5, FPDF.

### 3. Simulasi Pendaftaran BPJS
Website simulasi layanan publik untuk pendaftaran kepesertaan BPJS Kesehatan secara online.
* **Fitur**: Validasi formulir pendaftaran di sisi client dan server, upload berkas pendukung, dan tampilan kartu peserta sementara.
* **Teknologi**: HTML/CSS, JavaScript, PHP Native.

### 4. Monitoring Kualitas Air - Skripsi (IoT)
Sistem cerdas berbasis IoT untuk pemantauan kualitas air arwana (pH, suhu, kekeruhan) secara otomatis.
* **Fitur**: Dashboard monitoring real-time, visualisasi tren data menggunakan grafik Chart.js, dan implementasi algoritma **ANFIS** untuk kontrol aktuator otomatis.
* **Teknologi**: Python, Firebase, ESP32, Tailwind CSS.
